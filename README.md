# **ION Desktop app**

## Usage

### Install Dependencies

```bash
$ cd my-app

# using yarn or npm
$ yarn (or `npm install`)

# using pnpm
$ pnpm install --shamefully-hoist
```

### Use it

```bash
# development mode
$ yarn dev # (or `npm run dev` or `pnpm run dev`)

# production build
$ yarn build # (or `npm run build` or `pnpm run build`)
```
