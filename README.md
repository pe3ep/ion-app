# **ION Desktop app**

## Usage

### Clone the project

```bash
$ git clone https://github.com/pe3ep/ion-app
```

### Install Dependencies

```bash
$ cd ion-app

# using yarn or npm
$ yarn (or `npm install`)

# using pnpm
$ pnpm install --shamefully-hoist
```

### Use it

```bash
# development mode
$ yarn dev # (or `npm run dev` or `pnpm run dev`)

# production build
$ yarn build # (or `npm run build` or `pnpm run build`)
```
